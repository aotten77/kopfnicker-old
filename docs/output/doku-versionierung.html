Versionierung der Dokumentation

Die Dokumentation liegt sowohl auf Git als auch in der Knowledgebase. Workingdrafts/Revisionen bleiben der Knowledgebase vorenthalten und werden nur auf Git hinterlegt und profitieren somit von den Vorteilen der Git-Verisonierung. Erst bei Änderung der Nebenversionsnummer wird die Gesamtdokumentation und somit auch die Knowledgebase aktualisiert.
